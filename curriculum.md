# Учебный план: Go с нуля до продвинутого уровня

**Для Java/Spring разработчика**

---

## Основы

### Глава 1: Введение в Go
Установка Go, первая программа "Hello, World!", структура проекта. Отличия от Java: компиляция, отсутствие JVM.

### Глава 2: Переменные и базовые типы данных
Объявление переменных (var, :=), базовые типы (int, float, bool, string). Отличия от Java: type inference, отсутствие автоупаковки.

### Глава 3: Управляющие конструкции
if, for (единственный цикл в Go!), switch. Отличия от Java: switch без break, отсутствие while.

### Глава 4: Функции
Объявление функций, множественный возврат, именованные результаты. Отличия от Java: функции как first-class citizens.

### Глава 5: Массивы и слайсы
Массивы фиксированной длины, слайсы как динамические массивы, append, copy, slicing. Отличия от Java: array != ArrayList.

### Глава 6: Мапы (Maps)
Создание, добавление, удаление элементов, проверка существования ключа. Отличия от Java: встроенный тип, а не класс.

## Средний уровень

### Глава 7: Указатели
Что такое указатели, & и *, передача по значению vs по ссылке. Отличия от Java: явные указатели без арифметики.

### Глава 8: Структуры (Structs)
Определение структур, инициализация, встраивание структур. Отличия от Java: нет классов, есть композиция вместо наследования.

### Глава 9: Методы
Методы с value и pointer receiver, когда использовать каждый. Отличия от Java: методы вне структур.

### Глава 10: Интерфейсы
Определение интерфейсов, неявная имплементация, пустой интерфейс interface{}. Отличия от Java: неявная реализация, duck typing.

### Глава 11: Обработка ошибок
Возврат ошибок, error интерфейс, создание кастомных ошибок, паника и recover. Отличия от Java: нет exceptions, явный возврат ошибок.

### Глава 12: Пакеты и модули
Создание пакетов, импорт, go.mod, go.sum, семантическое версионирование. Отличия от Java: нет Maven/Gradle, простая структура.

### Глава 13: Defer, Panic, Recover
Отложенное выполнение (defer), паника, восстановление после паники. Отличие от Java: defer вместо finally.

## Конкурентность

### Глава 14: Горутины (Goroutines)
Создание горутин, легковесные потоки, go keyword. Отличия от Java: горутины vs Thread, минимальные накладные расходы.

### Глава 15: Каналы (Channels)
Создание каналов, отправка и получение данных, буферизованные каналы. Отличия от Java: встроенная примитива для коммуникации.

### Глава 16: Select
Оператор select для работы с несколькими каналами, timeout, default case. Отличия от Java: нет аналога.

### Глава 17: Sync пакет
Mutex, RWMutex, WaitGroup, Once. Сравнение с Java: synchronized, Lock, CountDownLatch.

## Продвинутый уровень

### Глава 18: Работа с файлами
Чтение, запись файлов, os и io пакеты, буферизация. Сравнение с Java IO/NIO.

### Глава 19: Работа с JSON
Маршалинг, анмаршалинг, теги структур, encoding/json. Сравнение с Jackson/Gson.

### Глава 20: Тестирование
testing пакет, table-driven tests, бенчмарки, coverage. Отличия от JUnit.

### Глава 21: HTTP клиент и сервер
net/http пакет, создание простого HTTP сервера, роутинг, middleware. Сравнение со Spring MVC.

### Глава 22: Context
Управление жизненным циклом запросов, отмена операций, таймауты. Отличия от Java: нет встроенного механизма.

### Глава 23: Работа с базами данных
database/sql, подключение к PostgreSQL/MySQL, CRUD операции, транзакции. Сравнение с JDBC/JPA.

### Глава 24: Generics (обобщённые типы)
Введение в generics (Go 1.18+), type parameters, constraints. Сравнение с Java Generics.

### Глава 25: Лучшие практики и паттерны
Идиоматичный Go код, code organization, распространённые паттерны проектирования в Go.
