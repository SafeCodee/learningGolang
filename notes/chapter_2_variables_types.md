# Глава 2: Переменные и базовые типы данных

## Объявление переменных

### 1. С помощью var

```go
var name string = "John"
var age int = 30
var price float64 = 99.99
var isActive bool = true
```

### 2. С type inference (вывод типа)

```go
var name = "John"      // Go сам определит тип string
var age = 30           // int
var price = 99.99      // float64
```

### 3. Короткое объявление := (внутри функций)

```go
name := "John"         // Самый частый способ
age := 30
price := 99.99
```

**Важно:** `:=` работает только внутри функций!

### 4. Множественное объявление

```go
var x, y int = 1, 2
a, b := 10, 20
name, age := "Alice", 25
```

## Базовые типы данных

### Целые числа

```go
var a int = 42           // Платформозависимый (32 или 64 бита)
var b int8 = 127         // -128 до 127
var c int16 = 32767      // -32768 до 32767
var d int32 = 2147483647
var e int64 = 9223372036854775807

var f uint = 42          // Беззнаковый
var g uint8 = 255        // 0 до 255 (byte — алиас для uint8)
```

### Числа с плавающей точкой

```go
var pi float32 = 3.14
var e float64 = 2.71828  // Более точный, используется по умолчанию
```

### Строки

```go
var greeting string = "Hello"
var multiline = `Многострочная
строка
в обратных кавычках`
```

### Булевы значения

```go
var isValid bool = true
var isComplete bool = false
```

## Нулевые значения (Zero Values)

В Go **все переменные имеют значение по умолчанию** (в отличие от Java, где примитивы имеют дефолты, а объекты — null).

```go
var i int        // 0
var f float64    // 0.0
var s string     // "" (пустая строка)
var b bool       // false
```

**В Java:** неинициализированные локальные переменные — ошибка компиляции.
**В Go:** всегда есть безопасное значение по умолчанию.

## Константы

```go
const Pi = 3.14159
const MaxUsers = 100

// Множественные константы
const (
    StatusOK = 200
    StatusNotFound = 404
    StatusError = 500
)
```

**Важно:** константы вычисляются на этапе компиляции.

## Type Conversion (преобразование типов)

В Go **нет автоматического приведения типов** (в отличие от Java).

```go
var i int = 42
var f float64 = float64(i)   // Явное преобразование
var u uint = uint(f)

// В Java было бы: double f = i; (автоматически)
```

## Отличия от Java

| Аспект | Java | Go |
|--------|------|-----|
| **Объявление** | `int x = 10;` | `x := 10` или `var x int = 10` |
| **Type inference** | С Java 10 (`var`) | Всегда был (`:=`) |
| **Нулевые значения** | Примитивы: дефолты, объекты: null | Всегда безопасное значение |
| **Автоприведение** | int → long → float автоматически | Только явное преобразование |
| **Константы** | `final` или `static final` | `const` |
| **Строки** | `String` (объект) | `string` (встроенный тип) |

## Важные моменты

✅ **`:=` — самый частый способ** объявления переменных
✅ **Нет null для примитивов** — всегда есть zero value
✅ **Явное преобразование типов** — безопаснее, но многословнее
✅ **string — встроенный тип**, а не класс как в Java
