# Первый промпт для Claude Code

Создай структуру для изучения Go с нуля:

## 1. Файл curriculum.md
Создай учебный план с главами от базового к продвинутому уровню.
- Я разработчик Java/Spring, но Go изучаю с нуля
- Хочу изучить ВСЁ: от базового синтаксиса до продвинутых тем
- В curriculum.md только НАЗВАНИЯ глав и краткое описание (1-2 предложения)
- БЕЗ полного контента - материал ты будешь давать в чате когда я запрошу
- Примерно 20-25 глав

Примерные темы:
- Базовый синтаксис, типы данных, переменные
- Функции, методы, структуры
- Массивы, слайсы, мапы
- Указатели
- Интерфейсы
- Error handling
- Горутины и каналы
- Пакеты и модули
- Тестирование
- Работа с файлами, JSON
- HTTP серверы
- Базы данных
- И другие важные темы

## 2. Файл progress.json
Создай файл для отслеживания прогресса между сессиями:
```json
{
  "current_chapter": 1,
  "completed_chapters": [],
  "last_session_date": "2025-10-22",
  "total_chapters": 25,
  "notes": ""
}
```

## 3. Папка exercises/
Создай пустую папку для практических заданий.

## Процесс обучения:
1. Каждую сессию я пишу: "Продолжаем, смотри progress.json"
2. Ты читаешь прогресс и даёшь материал текущей главы В ЧАТЕ
3. После объяснения создаёшь файл exercises/chapter_N_exercise.go с заданием
4. Я решаю задание сам
5. Показываю тебе код на проверку
6. Ты обновляешь progress.json

Начни с создания этих файлов.

ВАЖНО: В этой папке также будет файл CLAUDE.md с подробными инструкциями для тебя. Читай его каждую сессию.
